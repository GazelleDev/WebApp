You are an expert senior front-end engineer + brand designer. Build a production-ready marketing website for a new coffee shop called “Gazelle”.

GOAL
Create a high-end, modern, fast website that visually matches the attached store renderings: warm cream/beige interior, marble surfaces, brass/gold details, soft architectural curves/arches, subtle luxury, plants, clean lines, airy space. (See the PDF renderings: bright interior, golden accent lighting, arched feature wall with Gazelle mark, marble counters, wood seating.) Make the site feel premium, minimal, and calm.

TECH STACK (Replit-friendly)
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion for subtle animations
- next/image for optimized images
- next-sitemap (optional) OR create a simple sitemap.xml route
- No database needed
- Use a single environment variable for contact form email service ONLY if necessary. Otherwise implement a “mailto:” and a clean form UI.

SITE STRUCTURE
Create these routes:
1) "/" Home (hero + sections)
2) "/menu" Menu (beautiful, categorized menu page with placeholders)
3) "/location" Location & Hours (map embed + parking notes placeholders)
4) "/about" About (brand story + values)
5) "/gallery" Gallery (grid + lightbox)
6) "/contact" Contact (form + social links)
7) "/privacy" basic privacy policy placeholder
Add a top nav + footer on every page.

DESIGN SYSTEM (match the renderings)
Colors (Tailwind theme variables):
- Background: warm ivory / sandstone (#F6F1E8-ish)
- Surface: soft stone (#EFE6DA-ish)
- Text: deep espresso (#1D1A17-ish)
- Muted text: warm gray (#6B625A-ish)
- Accent metallic: soft gold/brass (#C8A25A-ish) used sparingly
- Line/border: subtle taupe (#D8CDBF-ish)

Typography:
- Headings: elegant serif (use a Google Font like "Cormorant Garamond" or "Playfair Display")
- Body: modern sans (use "Inter" or "Manrope")
Implement fonts with next/font/google.

Visual language:
- Rounded corners (2xl), soft shadows, subtle borders
- Large whitespace, calm layout, premium feel
- Use “arched” motifs: rounded top corners on some image frames/section containers
- Add subtle gold accent lines and icons; never overdo it
- Use gentle motion: fade/slide-in on scroll, micro-interactions on buttons
